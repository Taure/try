## Installation

Just add it to your ``rebar.config`` deps:

```erlang
  {deps, [
    ...
    {try.lfe.io, ".*",
      {git, "git@github.com:lfex/try.lfe.io.git", "master"}}
      ]}.
```

And then do the usual:

```bash
    $ rebar get-deps
    $ rebar compile
```
